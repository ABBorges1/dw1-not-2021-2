Elementos de bloco e de linha

Elementos de bloco

Sempre ocupam toda a largura disponível do seu elemeto pai. Quando colocados em sequência, ficam um embaixo do outro.

p

div

h1

blockquote
  1. ol

Elementos de linha

Ocupam apenas a largura necessária para o seu conteúdo e ficam lado a lado caso haja largura disponível.

a span em strong code abbr

Mudando o comportamento

Um elemento de bloco pode se comportar como um elemento de linha colocando, no CSS, display: inline

Da mesma forma, um elemento de linha pode se comportar como um elemento de bloco colocando-se, no CSS, display: block.
